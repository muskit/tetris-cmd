# tetris-cmd
Tetris for the Windows Command Prompt.

**NOTE:** it is highly recommended that you set your console font to "Raster Fonts" 12x16 *when you run the executable!*

## Controls

|Command           |Key          |
|:----------------:|------------:|
|Left              |LEFT ARW     |
|Right             |RIGHT ARW    |
|Clockwise         |[X] / UP ARW |
|Counter-Clockwise |[Z]          |
|Soft Drop         |DOWN ARW     |
|Hard Drop         |[SPACE]      |
|Quit              |[ESC]        |
|Debug Info Toggle |[F3]         |
